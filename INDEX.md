## How to get started

```bash
git clone https://github.com/Metaverse-Football-League/mfl-workspace.git ~/mfl-workspace
cd ~/mfl-workspace
./setup.sh
```

Then navigate to your domain's repo and start working:

```bash
cd mfl-marketing && claude
```
